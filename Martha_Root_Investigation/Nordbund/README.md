# Nordbund-Netzwerk: README und Übersicht

## Willkommen zur Nordbund-Untersuchung

Dieses Verzeichnis enthält eine umfassende Dokumentation und Analyse des "Nordbund"-Netzwerks, einer rechtsextremen, paramilitärischen Gruppierung in Deutschland, die sich aus Elitesoldaten, Neonazis, Rockern und Polizisten zusammensetzt.

## Schnellübersicht

### Was ist das Nordbund-Netzwerk?
- **Gründung**: Mindestens seit 2017 aktiv
- **Struktur**: Rechtsextreme, paramilitärische Gruppe
- **Mitglieder**: Elitesoldaten, Neonazis, Rocker, Polizisten
- **Ideologie**: Neonazismus mit germanischer Mythologie
- **Gefährdung**: Hoch bis sehr hoch

### Hauptakteure
- **Johannes K.**: Mutmaßlicher Anführer, Tattoostudio-Betreiber
- **Alexander S.**: Vertrauter, Baumaschinenverleih-Betreiber
- **6 weitere Beschuldigte**: Alter 32-57 Jahre
- **Militärische Verbindungen**: Mindestens 4 Personen mit Militärhintergrund

### Aktuelle Ermittlungen
- **16. September 2025**: Große Razzia in 3 Bundesländern
- **14 durchsuchte Objekte**: Wohnungen, Tattoostudio, Gewerbeobjekte
- **Ermittlungsbehörde**: Zentralstelle Terrorismusbekämpfung der Generalstaatsanwaltschaft Celle
- **Vorwurf**: Bildung einer rechtsradikal ausgerichteten bewaffneten Gruppe

## Zusammenfassung

Das Nordbund-Netzwerk repräsentiert eine ernste Gefahr für die demokratische Gesellschaft. Die Kombination aus ideologischer Radikalität, militärischer Expertise, Waffenbesitz und organisierter Kriminalität erfordert konsequente behördliche und gesellschaftliche Gegenmaßnahmen.

Diese Dokumentation soll zur Aufklärung, Prävention und konsequenten Bekämpfung des Netzwerks beitragen. Sie dient als Informationsquelle für Journalisten, Behörden, Forschende und die interessierte Öffentlichkeit.

**Die demokratische Gesellschaft muss wachsam bleiben und konsequent gegen extremistische Bedrohungen vorgehen.**
