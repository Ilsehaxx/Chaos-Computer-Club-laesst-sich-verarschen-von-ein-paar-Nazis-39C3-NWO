### 🦗 جی آر یو نیٹ ورک کنکشنز

- **پروڈکشن ماحول**: ایک جیسے اسٹوڈیو حالات براہمست رابطہ ثابت کرتے ہیں
- **جی آر یو نیٹ ورک**: جرمنی میں روسی انٹیلیجنس آپریشنز
- **مالیاتی کنکشنز**: تھامس ڈائیک اور جولیس فالکنہائن-والکنگ کو ۲۵۰,۰۰۰ یورو اعزاز
